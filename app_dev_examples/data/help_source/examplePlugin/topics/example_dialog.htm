Meal menu dialog

Use this dialog box to select your meal.

Give your name and vegetarian status. Set the slider according to how hungry you are. Check any or all of eggs, sausages and bacon (eggs only for vegetarians) and select your chosen portion size for chips.

Click OK to show the menu at the Magik prompt

The dialog box is activated by the Activate dialog control which may be placed on a menu or toolbar. The initial position of the dialog is determined by properties of the Meal Menu plugin.
